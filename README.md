# Air Quality Monitoring System

## 📌 Project Description
This project is an IoT-based Air Quality Monitoring System using ESP32 and MQ gas sensors. It detects harmful gases and provides real-time monitoring.

## ⚙️ Components Used
- ESP32 Microcontroller
- MQ Gas Sensor
- LCD Display
- Buzzer

## 📱 Features
- Real-time air quality monitoring
- Buzzer alert for unsafe conditions
- Blynk for remote monitoring
- ThingSpeak for data logging

## 🚀 How It Works
The MQ sensor detects gas levels → ESP32 processes data → LCD displays values → buzzer alerts if unsafe → data sent to Blynk & ThingSpeak.

## 📊 Platforms Used
- Blynk (Mobile Monitoring)
- ThingSpeak (Cloud Data Storage)
